# DreamStay Frontend Demo

DreamStay is a premium hotel and resort booking website demo built with React.js and Tailwind CSS.

This project is intentionally frontend-only for UI demonstration.

## Tech Stack

- React.js
- Tailwind CSS
- React Router
- Vite

## Included Pages

- Home Page
- Rooms Page
- Amenities Page
- Gallery Page
- Booking Page
- About Page
- Contact Page

## Features

- Luxury visual style (gold, white, dark blue)
- Mobile responsive layout
- Hero image slider and gallery slider
- Room filtering (type, price, guest capacity)
- Modern card layouts
- Smooth transitions and polished UI interactions
- Booking and contact forms (frontend behavior only)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This demo does not include a Node.js backend or MongoDB integration.
- Booking and contact submissions are displayed in the UI but are not persisted.
